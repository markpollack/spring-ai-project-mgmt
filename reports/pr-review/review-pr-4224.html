🎯 Executive Summary

Problem Being Solved

This PR adds support for Mistral AI's new Codestral Embed model to the Spring AI framework. The existing implementation only supported the mistral-embed model, and users needed access to the newer codestral-embed model which provides 1536-dimensional embeddings optimized for code embedding tasks.

Solution Quality Assessment

This appears to be a straightforward, well-structured addition following established patterns in the codebase. The use of parameterized tests demonstrates good testing practices, and the minimal scope reduces risk. However, the empty PR description and lack of linked issues significantly hurt the quality of the contribution from a documentation and traceability perspective.

Risk Summary

This PR adds support for the Mistral AI Codestral Embed model with low risk. The implementation follows Spring AI patterns correctly, includes comprehensive test coverage for both models, and uses safe hardcoded dimension mappings. Primary risk is the static nature of the dimensions map which could require code updates if Mistral AI changes model specifications.
